**Description**

NetPractice project is a general practical exercise designed to teach the basics of computer networking:
how to configure IP addresses, how to connect devices through a router, and the role of a gateway within a network.


**Resources**

Networking concepts studied:
- TCP/IP addressing
- Subnet masks
- Default gateway
- Routers and switches
